# Stock Explorer 📈

An investment learning tool built for self-directed research. Research stocks, understand price moves, compare companies, and map industry value chains.

## What it does

| Tab | What you can do |
|-----|----------------|
| **Why Did It Move?** | Enter any ticker + date range to get an AI explanation of what drove the price — earnings, news, macro events, analyst changes |
| **Company Research** | Full company profile — business model, financials, competitive position, risks, valuation |
| **Compare Stocks** | Head-to-head comparison on financials, valuation, moat, and risk |
| **Industry Chain** | Map any industry's full value chain from raw materials to end consumer, with a clickable flow diagram showing real branching paths |

## Getting started

### 1. Get an Anthropic API key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Create a free account
3. Add a small credit balance — **A$5–10 will last months** at typical usage (a full industry chain analysis costs ~3–5 cents)
4. Go to **API Keys** and create a new key (starts with `sk-ant-`)

### 2. Open the tool

Go to: **[https://alistairmccreadie-bot.github.io/stock-explorer/](https://alistairmccreadie-bot.github.io/stock-explorer/)**

On first visit you'll be prompted for your API key. Enter it once — it's saved in your browser on that device.

> **New device?** You'll need to enter your key again the first time on each device. The key is stored locally in your browser and never sent anywhere except directly to Anthropic.

### 3. Cost guide

| Action | Approx. cost |
|--------|-------------|
| Why Did It Move (1 month) | ~1–2 cents |
| Company Research | ~1–2 cents |
| Compare two stocks | ~1–2 cents |
| Industry Chain (full) | ~3–5 cents |

Check your usage any time at [console.anthropic.com/usage](https://console.anthropic.com/usage).

## Features

- **Persistent results** — your last analysis on each tab is saved in your browser. Come back later and it's still there. Click "Clear" to run a fresh analysis.
- **Interactive flow diagram** — click any stage in the Industry Chain to highlight what it feeds into and receives from
- **⭐ Beginner indicators** — the flow diagram marks 1–2 stages per industry as good starting points for new investors
- **Margin indicators** — each stage in the flow diagram shows indicative margin ranges
- **Date range analysis** — use quick-select buttons (1 Week, 1 Month, 1 Quarter, etc.) or pick custom dates
- **Drill down** — after exploring an industry chain, enter any ticker for a focused deep-dive on that specific company's role
